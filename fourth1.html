React Components

Components are the smallest, yet most fundamental part of React. They're similar, in concept, to things like widgets and modules.

These components include code which defines how DOM elements are created and how users can interact with them.

We're going to be referring to pure JavaScript as JS and the React version of JavaScript as JSX.

You can find the accompanying source-code, for this tutorial, at: https://github.com/formative/tutorial-react-components.
